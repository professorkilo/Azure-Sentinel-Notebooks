# Sentinel Lake Notebooks

### Overview

These notebooks are intended for use by customers of the Microsoft Sentinel Lake using the provided customer notebook environment.  These notebooks will rely on libraries and configurations that are preloaded in that environment.

[Sentinel Data Lake Overview](https://learn.microsoft.com/en-us/azure/sentinel/datalake/sentinel-lake-overview)

[Notebook Experience Overview](https://learn.microsoft.com/en-us/azure/sentinel/datalake/notebooks)

[Microsoft Sentinel Provider](https://learn.microsoft.com/en-us/azure/sentinel/datalake/sentinel-provider-class-reference)

<br>

### How to Run Notebooks

Currently notebooks are only supported from the Microsoft Sentinel extension within VS Code.  Refer to this document on how to install and use that extension to run notebooks: 

[Sentinel VS Code Extension](https://learn.microsoft.com/en-us/azure/sentinel/datalake/notebooks)
